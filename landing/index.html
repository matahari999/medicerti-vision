주요 기능

4가지 핵심 안전 감지

CCTV 영상 하나로 병원 내 모든 위험 상황을 자동으로 감지하고 즉시 알림을 전송합니다.

🫀

낙상 감지

YOLOv8-Pose ONNX 모델이 환자의 자세 변화를 실시간으로 분석하여 낙상을 0.2초 이내에 감지합니다. 오탐률을 최소화하는 다단계 검증 알고리즘 적용.

Pose Estimation YOLOv8 실시간
🚪

구역 이탈 감지

GeoFence 기술로 병동·ICU·수술실 등 제한 구역을 설정하고, 환자가 구역을 벗어날 경우 즉시 알림을 발송합니다. 다중 구역 동시 모니터링 지원.

GeoFence 다중 구역 즉시 알림
👁

장시간 배회 감지

ROI 트래커가 특정 구역에서 일정 시간 이상 머무는 인원을 감지합니다. 야간 병동 혼자 배회하는 환자를 자동으로 탐지하여 간호사에게 알립니다.

ROI Tracker 시간 임계값 야간 특화
🛡

미등록 외부인 감지

얼굴 인식 화이트리스트로 등록된 직원·보호자를 제외한 미등록 인원을 자동 감지합니다. 출입 통제 로그를 자동으로 기록하여 인증평가 서식으로 출력합니다.

얼굴 인식 화이트리스트 출입 통제
작동 방식

설치 3단계로 바로 시작

복잡한 클라우드 설정 없이 병원 PC에 설치하면 즉시 운영 가능합니다.

1

인스톨러 실행

119MB 단일 파일 설치 프로그램을 실행하면 AI 엔진·대시보드·모든 의존성이 한 번에 설치됩니다. Python, Node.js 별도 설치 불필요.

→
2

카메라 연결

기존 병원 CCTV의 RTSP 주소를 입력하거나 네트워크 자동 스캔 기능으로 카메라를 자동으로 찾아 연결합니다.

→
3

실시간 모니터링

브라우저에서 대시보드에 접속하면 즉시 AI 감지가 시작됩니다. Telegram으로 실시간 알림을 수신하고 PDF 보고서를 자동 생성합니다.
